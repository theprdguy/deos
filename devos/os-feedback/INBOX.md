# OS Feedback Inbox

OS-level friction/improvements collected from any project session.

> Append via `os3 feedback "..."` from any project, or add a `- {ISO-ts} [origin] text` line here directly.
> Reviewed at session start (devos/prompts/claude/session-start.md); ripe items are converted to tickets
> or swept during a periodic consolidation pass. One line per entry.
> This is the load-bearing pipe of the "improve the OS from product friction" operating model —
> if it stays empty while product work happens, the friction is evaporating into chat instead of being captured.
