# Architecture Decision Records (ADR)

> Precedence: `devos/CONTEXT.md` § Locked Decisions 가 ticket-facing SSOT 이며,
> 본 디렉토리는 long-form detail 보관소이다. ticket / reviewer 는 CONTEXT.md 의
> D-XX 를 enforce 하고, ADR-NNN 은 trade-off 분석 / 사유 archive 용으로 참조한다.

## Format
```markdown
# ADR-XXX: Title
Date: YYYY-MM-DD
Status: accepted | superseded | deprecated

## Context
Why this decision was needed.

## Decision
What was decided.

## Consequences
Trade-offs and implications.
```
